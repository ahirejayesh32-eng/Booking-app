# Booking App

A simple and clean full-stack booking application I built with Node.js, Express, and PostgreSQL.

## ✨ Features

- User registration and login
- JWT authentication
- Create, view, and manage bookings
- PostgreSQL database
- REST API endpoints
- Easy environment configuration

## 🛠️ Tech Stack

- **Backend**: Node.js + Express.js
- **Database**: PostgreSQL
- **Authentication**: JWT
- **Others**: dotenv for environment variables

## 🚀 Quick Start

### Prerequisites
- Node.js (version 16 or higher)
- PostgreSQL installed and running
